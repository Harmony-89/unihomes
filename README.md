 UniHousing (University of Embu House Hunting)

UniHousing is a specialized web platform designed to help students at the University of Embu find accommodation easily. It connects students directly with local landlords, providing a searchable and verified database of rentals near the campus.

## 🚀 Features

- **AJAX-Powered Search:** Fast, real-time filtering by property type (Bedsitter, One-bedroom, etc.), price range, and area (Gate A, Gate B, etc.) without page reloads.
- **Landlord Management:** Secure signup/login and a multi-step property upload form with live image previews.
- **Automated GPS Location:** Integrated "Use My Location" feature that captures precise GPS coordinates and generates a Google Maps navigation link for students.
- **Dynamic Image Gallery:** High-performance property details page where clicking thumbnails instantly updates the main display image.
- **Privacy First:** Landlord contact details (phone/email) are protected and only visible to logged-in students to prevent spam.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (AJAX / Fetch API)
- **Backend:** PHP 8.2
- **Database:** MySQL (Relational Schema)
- **Storage:** Local Filesystem with Persistent Railway Volumes
- **Hosting:** [Railway.app](https://railway.app)
